# CsGrafeq

## 简介

CsGrafeq是一项用c#实现的开源几何画板。出于性能和跨平台需要的原因，隐函数绘制再CsGrafeq暂时不支持。\
[旧版本项目](./OldVersion/README.md) 是专为隐函数制作的，这项功能将会在未来合并入CsGrafeq。\

## 特点
使用AOT(Ahead Of Time)发布, 无需安装 dotNET Core 运行时，无需携带大量动态链接库
